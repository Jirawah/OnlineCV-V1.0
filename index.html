Conception et développement d’un prototype complet d’application de gestion médicale (HealthCareProto), dans le cadre de la formation CDA.

Le backend repose sur une architecture microservices Spring Boot : configuration centralisée (Spring Cloud Config), découverte de services (Eureka), communication interservices (Feign), et sécurisation via JWT (Spring Security). L’ensemble est conteneurisé avec Docker et orchestré par Docker Compose.

Les briques applicatives développées :
• Microservice d’authentification et gestion des utilisateurs sécurisée (Spring Security + JWT).
• Microservice de gestion des patients (MySQL).
• Microservice de gestion des notes médicales (MongoDB).
• Microservice de scoring de risque diabétique (analyse via Feign).
• Frontend Angular stylisé avec Angular Material, gérant l’inscription, la connexion et la gestion des entités.
• Dockerisation complète des services et orchestration via Docker Compose.
